Welcome to Running Dashboard!

The web app that allows you to store and analyze your running data

How to get started

  1. Sign up (or log in if you are an existing user)
  2. Record your run as a gpx file using your phone or wearable device

    A reccomended gps recording app for android is gps logger

  3. Navigate to "Add new run" on the menu on the left in order to upload your gpx file:
